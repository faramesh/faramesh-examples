# Quickstart Examples

Get up and running with Faramesh in under 2 minutes.

## Examples

| File | Description |
|------|-------------|
| `quickstart_llm.py` | 2-minute demo with a real LLM (OpenRouter) — full LangChain agent + Faramesh governance |
| `quick_demo.py` | Quick demo of core Faramesh capabilities |

## Quick Start

```bash
pip install faramesh langchain
export OPENROUTER_API_KEY=your_key  # free model used — no cost
python quickstart_llm.py
```
