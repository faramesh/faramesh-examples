# Ecommerce Agent Examples

Real-world ecommerce AI agents governed by Faramesh.

## Examples

| File | Description |
|------|-------------|
| `refund_bot.py` | RefundBot — autonomous DTC ecommerce customer service agent for handling refund requests |

## Quick Start

```bash
pip install faramesh
python refund_bot.py
```
